# Coatue 的公開市場 CIO Jaimin Rangwalla 講得蠻清楚他們現在是怎麼看待AI市場，這基本上也反映在他們的投資組合操作

Coatue 的公開市場 CIO Jaimin Rangwalla 講得蠻清楚他們現在是怎麼看待AI市場，這基本上也反映在他們的投資組合操作

#未上市巨頭的空前規模與成長曲線

目前的 AI 浪潮展現了歷史上最陡峭的採用與營收成長曲線。Coatue 公開市場投資長 Jaimin Rangwalla 指出，在過去的科技週期中，非凡的上市（IPO）規模如 2012 年的 Meta（當時為 Facebook），其市值大約是 1000 億美元，在當時已是歷史上最受矚目的事件。然而在今天，未上市公司的價值在公開上市之前，就已經直接衝進全球前 25 名。

Jaimin Rangwalla 表示：「現在未上市公司在公開上市之前，就已經直接衝進全球前 25 名之列，這是個前所未有的時代。OpenAI 最近一輪的估值高達 8000 多億美元，Anthropic 上一輪也達到 3000 多億美元的高位，而 SpaceX 的估值更來到 1.25 兆美元。它們正以科技巨頭和 Mag 7 三分之一甚至二分之一的速度，達到 250 億美元的年化營收。」

特別是營收的增加速度，徹底顛覆了以往軟體（SaaS）領域的認知。以 Anthropic 公開談論的數據為例，12 月底時其營收為 90 億美元，隨後這個數字達到了 300 億美元，等於增加了 200 億美元。

Jaimin Rangwalla 透露：「它們每個月大約增加 100 億美元，幾乎每週增加 25 億美元。SaaS 領域中多數公司一整年的經常性營收（ARR）甚至都不到 25 億美元，而它們在一個星期內就增加了這個數字。」目前這兩家 AI 巨頭合計的規模已超越了 Salesforce、ServiceNow 等知名 SaaS 平台，且僅花了幾年時間便達成。

#從追隨GPU到追隨GW的架構轉變

隨著 AI 供應鏈的演變，Coatue 團隊重新調整了覆蓋科技主題的框架。兩三年前投資 NVIDIA 時，核心焦點在於追隨 GPU 的發展，思考支撐數百億美元 GPU 的基礎設施。但現在，投資視角已經切換。

Jaimin Rangwalla 強調：「我們現在甚至切換到了一個不同的視角，那就是『追隨GW（Gigawatts）』。GW幾乎是 AI 成長的原子單位，同時也是目前市場上最嚴重的短缺之一。我們正在釐清吉瓦的所有投入要素、如何縮短GW的交付週期、誰是GW的買家，以及誰在使用GW所產生的 token 產出。」

#供應鏈的極度緊繃與短缺經濟學

目前市場上供應緊繃的狀態持續超出預期。在記憶體領域，甚至出現了過去在科技產業從未見過的長期承諾。

Jaimin Rangwalla 指出：「在記憶體方面，現在已經能聽到持續到 2029 年、2030 年的供應協議與保證承諾。我從未見過這種程度的緊繃。」這反映出全球最大的兆元級買家並非盲目投資，他們在高度成長的同時投入數千億美元，顯示這並非短暫的週期性起伏。

在這個脈絡下，Coatue 提出了一個清晰的市場框架：「短缺的賣方」（Sellers of shortage）與「短缺的買方」（Buyers of shortage）。贏家是那些掌握有限產能、面對龐大需求的賣方，包括半導體公司、電力設施、記憶體、硬碟、光學元件、基礎設施以及勞動力公司，甚至包含正在調升價格的 TSMC。

由於價格是推動營收成長的主要槓桿且固定成本固定，他們的營運利潤正以營收增幅的數倍速跳升，許多公司的獲利在短短一兩年間成長了 5 到 7 倍。

相反地，市場正在懲罰那些短缺的買方，也就是將大量資本支出（CapEx）投入建置的雲端巨頭（如 Microsoft、Amazon、Meta）。

Jaimin Rangwalla 解釋道：「買方因為資本支出過高，近期的現金流因而被吞噬，而這些資金直接轉移到了短缺賣方的口袋裡，因此賣方在目前大獲回報。如果記憶體價格上漲 100%，Microsoft 必須花費兩個單位而不是一個單位的資金，他們實際上並沒有獲得兩個單位的效益，他們獲得的效益跟之前價格低的時候是一樣的。」

不過，Google 與 Amazon 屬於特殊的混合陣營，因為 Google 擁有自研的 TPU，Amazon 也有自研的 Trainium，他們既是買方，同時也是短缺技術的賣方。

#Token經濟學與AI營收的真實來源

對於外界好奇 AI 公司的營收究竟從何而來，Jaimin Rangwalla 拆解，主要是企業、個人與專業消費者（prosumers）為了提升效率與生產力而付費。

頂尖的工程師和開發者表示，他們的生產力提高了 2 到 3 倍，使企業能夠執行更多原本受限於人力而無法開展的專案。這也促使企業重新想像 IT 預算在硬體、軟體及服務之間的配置。而這一切的核心就是 token。

Jaimin Rangwalla 說明：「基本上一顆 token 就是 AI 模型思考的單位。不論你何時提出問題、得到解答，系統所產生的 token 就是你看到的答案。這可以被視為一個智慧單位。

當我們談論 token 經濟時，核心其實是 agent 的普及以及 AI 所做的決策，而這些決策可以被具體量化為 token 數量。這與人類智慧很難被量化成一個單位非常不同。」

#Agent革命與OpenClaw的工作模式顛覆

相較於早期 ChatGPT 需要人類頻繁互動（human in the loop）並在中途停頓確認的模式，新一代技術迎來了巨大的突破。尤其是去年底 Opus 4.5 推出後，agent 的能力大幅釋放。

Jaimin Rangwalla 強調：「一個 agent 現在可以衍生出自己的子 agent，這增加了工作的深度、時間與品質。Agent 啟動 agent 是目前最被低估的突破之一。你可以直接交給 agent 一個專案然後離開，回來時它就已經推進得非常深了。」甚至可以指示系統衍生出另外 10 個 agent 來檢查前面所有 agent 的工作，讓人類不再成為工作速度的限制因素。

在這個趨勢下，名為 OpenClaw 的技術架構正在快速成形。這是一種新型態的「控制架構」（harness），改變了人類與模型互動的方式。它允許使用者透過一邊的電腦發送指令，控制另一端的實體或虛擬電腦採取行動。例如分析師可以透過 WhatsApp 傳訊息給 OpenClaw，讓它在背景處理訂餐廳、發送日曆邀請、安排接送車輛等一連串複雜任務。

目前包含 ByteDance、Tencent、Alibaba 在內的公司都在發展自己的 claw 概念。未來手機將成為控制這些 agent 的遙控器，而某個應用程式將演變成整合一切的超級 App。

#架構翻轉：CPU 需求大爆發與數位人口擴張

過去記憶體的需求主要與智慧型手機和電腦的普及率及硬體規格成長掛鉤。但在 AI 時代，個人所佔用的半導體與技術足跡正迎來 10 倍的擴張。以 Claude code 創辦人 Boris Cherny 的日常工作模式為例，他白天執行數個 agent，下班回家路上則啟動上千個 agent 讓它們通宵工作。這意味著每個人驅動的虛擬裝置數量可能高達數千個，每一個都需要 CPU、GPU、記憶體與電力支援。

這種行為轉變正在劇烈改變 AI 架構。過去的聊天機器人存在健忘症，每次對話都需要重新餵資料；但隨著新型加速器的演進，agent 開始具備持久記憶（persistent memory）。同時，CPU 的需求正迎來爆發。

過去 3 到 4 年中，架構大多是 1 顆 CPU 配 8 顆甚至 16 顆 GPU，計算全在 GPU 上完成。但現在，這個比例已經翻轉為 1 顆 CPU 配 4 顆 GPU，甚至未來有可能演變成 1 顆 GPU 搭配 4 顆或 8 顆 CPU。這是因為許多 agent 執行的任務屬於序列處理（serial processing），極度依賴 CPU。

Jaimin Rangwalla 指出：「這就像是數位人口的擴張。如果我們將全球人口乘以一千倍，這些數位人口都需要序列處理、平行運算和記憶體，就像每個人類一樣。

我們認為這個配比甚至有機會往反方向翻轉，變成 1 顆 GPU 搭配 4 顆 CPU，有些人甚至非常激進地認為可能會達到 1 顆 GPU 搭配 8 顆 CPU。」

在 CPU 領域，主要球員包括 Intel、AMD 和 ARM。而過去多年在半導體領域被視為落後者的 Intel，在迎來優秀的執行長以及產業趨勢轉變後，展現了極大的追趕潛力。

Jaimin Rangwalla 提出他的投資論點：「有些最好的投資論點往往非常簡單。當看到配比從四比一變成一比四，這在數學上代表了市場規模 16 倍的潛在改善；

再加上 Elon Musk 對 Intel 的認可並加快創新步伐，這兩者的結合對於這檔股票而言意義重大。儘管它今年有所表現，但相比其他半導體公司過去幾年 5 到 10 倍的漲幅，它依然是個落後者。」

#AI產業的潛在風險與技術顛覆

談及未來的潛在風險，Jaimin Rangwalla 認為最核心的是技術典範轉移。例如去年出現的模型技術突破，如果有人研發出能以更少電力、更少半導體和記憶體完成相同運算的架構，這短期內會衝擊現有的短缺供應商。

Jaimin Rangwalla 表示：「如果像去年發生的 DeepSeek 時刻那樣，又有另一個類似的突破出現，有人研發出能以更少電力、更少半導體和記憶體完成所有運算的模型，這是我作為一個每天面對市值投資組合的人會思考的風險。」

不過他進一步指出，這符合傑文斯悖論（Jevons' paradox）的邏輯：技術變得越便宜，人類就會找到越多元、越具創造力的方式去使用它，反而會加速 AGI 與機器人等領域的落地。在監控資料中心建設與市場瓶頸時，真正的挑戰在於瓶頸無處不在，包含發電、輸配電、NAND、DRAM、光學元件與建造資料中心的勞動力。目前市場高度聚焦在攫取這些賣方市場的短期利潤，但未來的某個時間點，權力結構將會發生轉移。
