# 這波cycle為何memory module houses毛利率這麼高？

原文連結: https://www.facebook.com/permalink.php?story_fbid=pfbid02tfezN7nGc5ePizBndGoXKKzgUPExeGAsGWaj2MxfScy7BoCavfkPGkGWMiYgVWg8l&id=100054201473657
這波cycle為何memory module houses毛利率這麼高？

1.這波cycle產品價格上漲的速度、斜率、幅度，歷史僅見

進貨->生產->庫存->通路銷售，有時間落差，當漲價斜率幅度速度，歷史最大的時候，銷售價(時間t+1)和進貨價(時間t)之間的價差，自然是是歷史最大

4Q25~1Q26~2Q26漲價幅度150%～250%，絕對是歷史最高最快，理由去年講過，因為CSP首度成為主力買家，背後原因AI不需再講

2.這波supply and demand的Gap，不確定是否史上最大，但一定數一數二大的Gap，史上缺貨最嚴重的前一兩次

雖然memoey module houses的進入障礙遠遠不如memory makers那麼大，長時間如此的話，一定有更多modules houses進入者進來稀釋該產業超額利潤，但畢竟需要期間，尤其通路布局需要時間，所以單一一次cycle的時候，module houses是固定的一批人，掌握貨源的一批人，你會不會加價賣？你不加客戶也會主動加價搶貨，除了第1點的時間落差之外本來就有利潤之外的加價賣？

3.上兩點是合約vs合約、現貨vs現貨，第3點是說同一個時間點之下的現貨價vs合約價Gap落差，可能也是史上最大，4Q25~1Q26

誰賣contract price？誰賣spot price？誰買contract price？誰買spot price？有沒有政府規定？沒有，有沒有機構認證？沒有？有沒有原廠規定？沒有，但如果你大量向原廠買contract price卻大量用spot price賣出去的話最好不要讓原廠知道，或者當關係好module houses在cycle downward trend的時候要配合原廠塞貨庫存或pull in PO讓財報平準化，這時候原廠可能也是睜一隻眼閉一隻眼

Spot market一般只佔5%，contract market佔95%，但spot和contract中間不是一刀切而是連續的，spot price和contract price之間不是一刀切跳躍的而是連續的數字，尤其在gap史上最大的時候，沒有人 "規定" 只能買賣contract price或跳一倍到spot price(最高點量極少)，中間的價格是連續的

memory module houses只要進貨多偏向、傾向一些contract price、銷售時多偏向、傾向一些spot price，就有可觀價差了

早年，例如NB大客戶memory是buy and sell料件，是品牌客戶議價採購，當年還有10%~20% local king brands(印度印尼歐洲等)和歐美通路商品牌如Walmart牌NB，太小不能向原廠buy and sell，授權ODM採購，這時候一樣，spot price和contract price的gap，同廠商同料號的part，有價差，不論upcycle或downcycle誰高誰低(現在local king死光幾乎沒有這種情況了)

4 module house的研發、品牌投資造成價值提升和毛利率提高，當然有，但這是經年累月、努力研發、努力行銷累積而成，慢慢創造競爭優勢，一點一點的慢慢提升毛利率，不可能半年內因為研發和品牌經營讓毛利率增加幾十%，否則半年前豈不是全公司都在睡覺？

以上講的都是cycle下的正常狀況，就可以創造可觀毛利率，還不包含如果module houses展現個別眼光、資訊、經驗、企業精神之下的刻意行為，在cycle中可能創造更多的毛利率
