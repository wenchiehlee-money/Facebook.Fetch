# 🚀 Perplexity CEO：AI 基礎設施瓶頸轉移，誰掌握瓶頸，誰就掌握獲利能力

原文連結: https://www.facebook.com/reel/1017908441194701/
🚀 Perplexity CEO：AI 基礎設施瓶頸轉移，誰掌握瓶頸，誰就掌握獲利能力
🔹HBM 記憶體仍是 AI 發展最大的瓶頸
Aravind Srinivas 認為，未來 6～12 個月，提供 HBM（高頻寬記憶體）的 Micron Technology，市值甚至可能比 Meta Platforms 更具成長潛力。

他指出，雖然 HBM 價格已大幅上漲，市場也認為美光股價已反映不少利多，但真正的關鍵在於 HBM 仍是 AI 產業最重要的供給瓶頸之一。只要供不應求的局面持續，供應商就擁有較強的定價能力，獲利仍有進一步提升的空間。

🔹AI Agent 興起，CPU 再次成為關鍵資源
隨著 AI Agent 開始執行更多實際工作，例如下載大量資料、整理檔案、資料轉換、執行程式及部署網站等，這些工作大多不是由 GPU 完成，而是交由 CPU 負責。

簡單來說：
1.GPU 負責 AI 模型推論並產生 Token。
2.CPU 負責執行 Agent 的工作流程（Agent Loops、Agent Harnesses）。
3.AI Agent 使用 CPU 的時間，未來甚至可能超過真人使用者。

因此，企業級 CPU 的需求正快速提升。

🔹AMD、Intel 有望受惠 CPU 需求回升
由於 CPU 再度成為 AI 基礎設施的重要瓶頸，Aravind Srinivas 認為，Advanced Micro Devices（AMD）與 Intel 都有望受惠於企業級 CPU 需求成長。

AI 的發展已不再只是 GPU 的競賽，而是整個運算架構都需要同步升級。

🔹真正有價值的是掌握瓶頸的供應商
Aravind Srinivas 提出一個核心觀點：

「誰是瓶頸，誰就能掌握定價權。」

目前 AI 基礎設施仍面臨供給不足，因此具備關鍵供應能力的企業，包括：
▪ HBM 記憶體供應商。
▪ SSD 儲存供應商。
▪ CPU 供應商。
▪ 其他 AI 基礎設施核心零組件廠商。

這些企業都有機會比單純興建資料中心的公司更具投資價值，因為真正稀缺的是關鍵硬體，而不是機房本身。

🎯結論
Aravind Srinivas 認為，AI 投資的重點正逐漸從「誰擁有最大的 AI 模型」轉向「誰掌握 AI 基礎設施的瓶頸」。當 HBM、CPU、SSD 等關鍵元件仍供不應求時，相關供應商將持續享有較高的定價能力與獲利成長潛力，也可能成為這一波 AI 浪潮中最大的受益者。
