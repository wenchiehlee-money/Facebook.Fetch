🔹 關鍵技術：高效率 P1 + P3 雙馬達架構
1、P1（發電機）
→ 專職發電，讓引擎維持在最佳效率區間

2、P3（驅動馬達）
→ 直接驅動車輪，提供主要動力來源

👉 屬於 串聯為主、並聯輔助（Series-Parallel Hybrid）
👉 多數時間以「電驅動」運作

🔹 為何這套架構被視為最先進油電之一？
1、以「電動車邏輯」重構油電系統
→ 馬達成為主體，引擎轉為能源供應角色

2、沒有傳統多段變速箱（AT / CVT）
→ 不靠換檔傳遞動力，而是由馬達直接驅動
→ 僅在高速時透過「單一固定齒比 + 離合器」直連引擎
👉 結構更簡單、機械損耗更低

3、雙馬達（P1 + P3）高度分工
→ 發電與驅動各自最佳化，效率最大化

總結:
👉 不只是傳統 Hybrid，而是「電驅主導的混合動力系統」
