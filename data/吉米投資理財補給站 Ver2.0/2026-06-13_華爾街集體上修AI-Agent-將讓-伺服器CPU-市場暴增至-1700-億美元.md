# 🚀 華爾街集體上修！AI Agent 將讓 伺服器CPU 市場暴增至 1,700 億美元

原文連結: https://www.facebook.com/reel/4382691318713819/
🚀 華爾街集體上修！AI Agent 將讓 伺服器CPU 市場暴增至 1,700 億美元
🔹 華爾街全面上修 伺服器CPU市場預測
美國銀行分析師 Vivek Arya 最新指出，全球伺服器 CPU 市場規模（TAM）有望從 2025 年的 350 億美元，大幅成長至 2030 年超過 1,700 億美元，較先前預估的 1,250 億美元進一步上修。

這不只是美銀的看法，近期包括 UBS、Citigroup 與 Morgan Stanley 等華爾街機構，也紛紛調高未來五年的 CPU 市場規模預測，顯示市場開始重新評估 CPU 在 AI 時代的重要性。

🔹 代理式 AI 成為最大成長引擎
過去兩年市場焦點幾乎都集中在 GPU，因為大型語言模型的訓練與推理主要依賴 GPU 算力。然而隨著 Agentic AI（代理式 AI）快速發展，CPU 的角色正變得愈來愈關鍵。

當 AI 助理幫使用者搜尋郵件、安排會議、更新行事曆、查詢企業資料或執行跨系統工作流程時，背後大量的任務調度、資料管理與系統控制工作，其實主要是由 CPU 負責完成。

換句話說，GPU 負責運算與生成內容，而 CPU 則負責執行與協調各項任務。未來 AI Agent 使用量愈高，對 CPU 的需求也將同步增加。

🔹 AI 基礎設施正在進入新階段
摩根士丹利認為，AI 基礎設施正在從過去的 GPU 主導模式，逐漸轉向 CPU、記憶體、網路與加速器共同協作的架構。

這代表未來資料中心的投資方向不再只是增加 GPU 數量，而是整體系統效能的提升。因此除了 GPU 之外，CPU、DRAM、HBM、高速網路設備以及先進封裝等領域，都有機會同步受惠。

市場過去認為 AI 是單純的 GPU 故事，但未來更可能演變成整個資料中心全面升級的故事。

🔹 哪些公司可能受惠？
在傳統伺服器 CPU 領域，Advanced Micro Devices 與 Intel 仍然是主要受惠者。

AMD 近年透過 EPYC 系列產品持續搶占資料中心市占率，而 Intel 雖然面臨競爭壓力，但若整體市場規模大幅擴張，仍有機會受惠於產業成長紅利。

另一方面，Arm Holdings 架構也正受到大型雲端業者青睞，因其具備較佳的能耗效率。包括亞馬遜、Google 等業者近年皆積極發展 ARM 架構伺服器。

此外，NVIDIA 已透過 Grace CPU 進軍 CPU 市場，而 Qualcomm 也傳出將推出新一代資料中心 CPU，顯示未來競爭將更加激烈。

🔹 CPU 可能是 AI 時代被低估的機會
目前市場對 AI 的關注仍集中在 GPU 與 AI 加速器，但如果代理式 AI 真的如華爾街預期般快速普及，CPU 很可能成為下一波最重要的受惠環節之一。

因為每一個 AI Agent 的背後，不只需要 GPU 進行推理，更需要大量 CPU 負責協調、管理與執行任務。當 AI 從聊天機器人進化成真正能替人工作的數位員工時，CPU 的需求成長速度可能遠超市場目前的預期。

🎯 結論
AI 產業正在從「GPU 時代」逐步走向「系統時代」。

未來的競爭不再只是比誰擁有更多 GPU，而是比誰能打造更完整的運算平台。隨著代理式 AI 加速普及，CPU 的重要性正在快速提升，而這也是華爾街近期集體上修伺服器 CPU 市場規模預測的核心原因。

如果相關預測成真，2030 年全球伺服器 CPU 市場規模突破 1,700 億美元，將代表 CPU 不只是 AI 的配角，而是下一輪 AI 基礎建設擴張的重要主角。
