🧠 AI 下一個容易被忽略的大市場：Memory Interface（記憶體介面晶片）
市場都在關注 GPU、HBM、ASIC，但 AI 伺服器還有一個高速成長的市場正在崛起──Memory Interface（記憶體介面晶片）。

最新產業報告預估，全球 Memory Interface 市場規模將在 2030 年達到 200 億美元，其中 MRDIMM 架構所需的 MRCD 與 MDB，預計將貢獻約 73% 的市場規模，成為 AI 基礎建設的重要受惠領域。

① Memory Interface 為什麼重要？
AI 模型越來越大，GPU 數量越來越多，代表記憶體必須提供更高頻寬、更低延遲與更大容量。

這時，Memory Interface 晶片就扮演關鍵角色，負責協調 CPU、記憶體控制器與 DDR5/MRDIMM 之間的高速資料傳輸，確保整個系統穩定運作。

② MRCD、MDB 是什麼？
▪ MRCD（Multi-Rank Clock Driver）：負責同步多個 Rank 的時鐘訊號，提升高速 DDR5 的穩定性。
▪ MDB（Multiplexed Data Buffer）：負責資料緩衝與高速傳輸，大幅提升 MRDIMM 的頻寬表現。

隨著 AI 伺服器開始導入 MRDIMM，MRCD 與 MDB 的需求也將快速成長。

③ 哪些公司值得關注？

🇺🇸 Rambus
▪ Memory Interface 技術領導廠商。
▪ 布局 MRCD、MDB、DDR5/DDR6 PHY IP。
▪ 是目前最純的 Memory Interface 概念股之一。

🇺🇸 Marvell Technology
▪ 透過收購 Inphi，擁有高速互連與部分記憶體介面技術。
▪ 雖然主要成長動能仍來自 AI 網路與客製化 ASIC，但也具備相關技術布局。

🇯🇵 Renesas Electronics
▪ 承接原 IDT 的 DDR5 記憶體介面產品線。
▪ 提供 Clock Driver、SPD Hub 等記憶體介面元件，是伺服器記憶體生態系的重要供應商。

📌觀察
AI 基礎建設的受惠者，不只有 GPU 與 HBM。

Memory Interface（記憶體介面晶片）正逐漸成為另一個高成長市場，而 MRCD 與 MDB 更是 MRDIMM 架構的核心元件。

如果 AI 伺服器持續朝向更高頻寬、更大容量發展，除了大家熟悉的 GPU 與 HBM 外，Rambus、Marvell、Renesas 等具備相關技術布局的公司，也值得持續追蹤。 #RMBS
