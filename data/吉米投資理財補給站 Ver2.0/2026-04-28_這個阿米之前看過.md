# 這個阿米之前看過~

原文連結: https://www.facebook.com/permalink.php?story_fbid=pfbid02Yo7wrToCinLe8Yzopg5WHQvaudNHX6gBFsNpj5csv1S1ECRXhn4hztCxiFagmbzdl&id=61565088683715
這個阿米之前看過~
「原油分餾（Fractional Distillation）」，本質就是把一鍋混在一起的碳氫化合物，依照「沸點不同」一層一層分離出來。
▪ 原油不是單一物質，是「各種長短不同的碳鏈混合物」
▪ 越短的碳鏈 → 沸點低 → 跑到塔頂
▪ 越長的碳鏈 → 沸點高 → 留在底部

🏗️ 分層結構（由上到下＝由輕到重）
🟢 1. LPG（< 40°C）
▪ 像瓦斯（丙烷、丁烷）
▪ 用途：瓦斯爐、燃料
▪ 最輕、最容易揮發

🟡 2. 石腦油（Naphtha 70–100°C）
▪ 石化產業最重要原料之一
▪ 拿去裂解 → 變塑膠、化學品
👉 塑化產業命脈

🚗 3. 汽油（40–150°C）
▪ 汽車燃料
▪ 需求直接連動經濟活動
👉 景氣好 → 開車多 → 汽油需求↑

✈️ 4. 煤油（150–250°C）
▪ 飛機燃料（Jet fuel）
▪ 也可供暖
👉 航空業指標

🚛 5. 柴油（250–350°C）
▪ 卡車、工業機械
▪ 物流＋製造業命脈

👉 柴油需求強 = 工業活動熱

🏭 6. 重油（350–450°C）
▪ 工業燃料、船舶
▪ 汙染較高
👉 現在逐漸被限制（ESG影響）

🛢️ 7. 瀝青（>450°C）
▪ 道路鋪設
▪ 最重、最黏
👉 幾乎「剩料再利用」

🔄 流程你要這樣記
原油進爐子加熱🔥
蒸氣往上跑
遇冷凝結在不同高度
每一層抽出不同產品

▪ 越長的碳鏈 → 沸點高 → 留在底部

👉 所以整個分餾塔就是：
底部最熱、頂部最冷
