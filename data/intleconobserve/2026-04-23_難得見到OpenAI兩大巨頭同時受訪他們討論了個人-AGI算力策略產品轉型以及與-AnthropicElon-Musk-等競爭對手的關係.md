難得見到OpenAI兩大巨頭同時受訪，他們討論了個人 AGI、算力策略、產品轉型，以及與 Anthropic、Elon Musk 等競爭對手的關係。

#個人AGI才是終點

Altman 指出，目前的模型雖然強大，但對使用者生活的認知仍極為有限。「你還是得不斷哄著它、調整提示，才能得到你要的東西。我們距離一個真正了解你所有 context 的模型，已經不遠了。」

Brockman 將這個願景稱為「個人 AGI」。「這個 AI 真的認識你，有你的 context，你可以信任它。當你問它財務或健康相關的問題時，它能給你值得信任的資訊。」他認為工作用 AI 與生活用 AI 之間的界線會變得模糊，兩者在技術底層其實是同一個 neural net。

Altman 舉例：「也許它知道你喜歡某位音樂家，那位音樂家剛好來你的城市，它主動注意到票開賣了，有很便宜的好位子，它就直接幫你買了。」

#從model到agent平台的轉型

Brockman 近期接手 OpenAI 產品方向的整合。他指出，軟體工程領域已經明顯從「編輯器旁的側邊欄」轉向 Codex 這類 agent 管理平台。「agents 會處理所有細節，處理所有瑣碎工作。」

這個轉變改變了模型與產品的關係。「模型已經從『就是產品本身』變成『產品的一部分』。以前模型上面只有很薄的一層軟體，但現在那一層變得很厚，包含 skills、connectors、computer use 怎麼接、context 跟 memory 怎麼管理。模型像是大腦，而我們正在打造身體。」

Sora 為何被砍？Brockman 解釋：「Sora 是技術樹的另一個分支，驅動 Sora 的模型跟核心 GPT 系列並未統一。」

#算力是利潤中心，不是成本中心

針對外界認為 OpenAI 在基礎設施上「踩煞車」的說法，Altman 直接否認：「我們會盡可能持續擴建算力。」

Brockman 補充了一個關鍵觀點：「算力對我們來說不是成本中心，而是利潤中心。

我們的商業模式其實非常簡單，就是租或買算力，然後以一定的利潤率轉售出去。只要利潤率是正的，它就可以無限擴展，因為需求是無限的。」

#美國在硬體上的困境

被問到美國是否在硬體製造上已經落後中國，Altman 坦承問題嚴峻。

「如果你要挑一件事讓美國在製造業上具備競爭力，答案是我們需要大量能製造更多機器人的機器人。但我們連一個 actuator 都做不出來。」

他認為機器人加 AI 是唯一的追趕路徑：「如果我們能做出機器人版的 Codex，你可以對它說『去配置一條這樣的產線』、『去搞清楚怎麼開採與精煉這種物質』，那局面就會改變。」

#三種未來與算力平權

Altman 描述了三種可能的未來。第一種是地板大幅抬升，每個人都「主觀上富有十倍」，但也會出現十個 trillionaire，不平等加劇；第二種是地板抬升幅度較小，但不平等也較低。

Brockman 提出第三條路：「AI 就是機會，但前提是你有 access、你有算力。無論你多會用 agents，沒有算力就跑不動。」他觀察到現在的孩子使用 agents 的能力會是他這一代的十倍，「如果你能把算力給到每個孩子，那這會是美國夢最極致的版本。」

Altman 補充：「每個人都應該希望有更多算力、更多基礎設施，以及盡可能便宜的 AI 取得管道。否則算力有限、供需失衡推高價格、只有富人能用，就會惡化不平等。」

#對Anthropic與Mythos的回應

Brockman 對 Anthropic 的執行力給予肯定：

「我們較晚意識到不能只建構抽象上擅長 coding 的模型。雖然我們在 coding 競賽上一直都有最好的數字，但你還得把模型應用到混亂的 repo、真實世界的資料上。這點 Anthropic 比我們更早體認到。」

但針對 Anthropic 近期以 Mythos 模型主打的 cyber security 敘事，Altman 語氣轉為直接：「有些人長期以來就想把 AI 控制在一小群人手中。恐懼行銷大概是合理化這件事最有效的方式。這就像在說：我們造了一顆炸彈，我們要丟在你頭上，我們可以賣你一個一億美元的避難所，但前提是我們挑你當客戶。」

#ElonMusk訴訟是說出真相的機會

針對即將到來的 Elon Musk 訴訟，Brockman 透露了當年決裂的關鍵時刻。當時 Sam、Ilia、Greg、Elon 所有人都同意 OpenAI 必須轉為營利組織才能推進使命，但談判陷入僵局。

「Elon 說他需要多數股權、需要當 CEO、需要完全控制權。我們幾乎要答應了。但絕對控制權這件事，即使他說未來會稀釋，讓我們停下來問：我們真的在乎讓這項技術造福所有人的願景嗎？不應該有一個人掌控整個未來，不管那個人是誰。那就是破局點。」

Brockman 表示：「多年來我們從未講過這個故事，但現在我們會講。我唯一擔心的是他在開庭前撤案，讓我們沒機會把話說清楚。」
