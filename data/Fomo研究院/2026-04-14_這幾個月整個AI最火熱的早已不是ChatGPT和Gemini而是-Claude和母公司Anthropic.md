# 這幾個月，整個AI最火熱的早已不是ChatGPT和Gemini，而是 Claude和母公司Anthropic。

原文連結: https://www.facebook.com/permalink.php?story_fbid=pfbid0WZvWrX1NrZfgRR1T8zWb3tmcHqZdaecWjHu8mHSEhBhog8xoyAtxLcWbNXHU3ExEl&id=61573146584049

這幾個月，整個AI最火熱的早已不是ChatGPT和Gemini，而是 Claude和母公司Anthropic。
​
為什麼？因為根據最新流出的數據，Anthropic 的年化經常性收入（ARR）已經突破 300 億美元，正式超車 OpenAI 的 250 億美元，成為全球營收最高的 AI 獨角獸。
​
同時，市場傳出他們正評估在今年下半年進行 IPO，預計募資超過 600 億美元，估值將推升至3800 億美元。
​
在最新一期的 商業周刊（商周.com） 專欄中，我針對 Anthropic 進行了完整的深度拆解，這裡我想先分享其中一小部份的觀察。
​
▋憲法式 AI：給機器寫一本人格說明書
​
你有沒有想過，為什麼跟 Claude 聊天的感覺跟 ChatGPT 或 Gemini 不一樣？為什麼這個看似「不夠有創意」、「動不動就拒絕你」的 AI，能在短短一年內在商業市場上把 OpenAI 逼到跳腳？
​
這一切，都要從它那本與眾不同的「人格說明書」說起。
​
Claude 的性格差異，不是那種「這個比較聰明」的差異，而是刻意設計的結果。Anthropic 用的方法叫做 Constitutional AI（憲法式 AI）。
​
其他公司如 OpenAI、Google 都用 RLHF（人類回饋增強學習），找一堆人類標註員來評分。
​
問題是，人類下意識會獎勵那些「拍你馬屁讓你開心」或「聽起來安全但其實沒用」的回答。
​
Anthropic 的解法是：給 AI 寫一本長達 84 頁的「憲法」，包含解釋、理由、案例與哲學討論，然後讓 AI 自己反覆閱讀、自我批評、自我修正，直到把這些價值觀內化。
​
這套憲法建立了一個鐵律般的優先順序：安全 > 道德 > Anthropic 的規定 > 對用戶有幫助。
​
這讓 Claude 變得不太拍馬屁，會問問題確認你到底要什麼，而且拒絕你的機率比其他 AI 高。在消費者市場，這可能是個缺點（這也是為什麼 ChatGPT 的月活用戶仍稱霸全球）；但在企業市場（B2B），這卻是致命的吸引力。
​
金融業、醫療業、政府機構要的不是「最有創意」的 AI，而是「最可靠、最不會出包」的 AI。Claude 的高拒絕率、一致的道德判斷、不亂講話的特性，對這些客戶來說是最大的賣點。
​
▋營收之戰：300 億美金的「數字遊戲」？
​
這套「安全至上」的邏輯，成功轉化成了商業回報。
​
Anthropic 的年化營收能衝破 300 億美元，靠的就是企業客戶毫不手軟的投資。目前，年花費超過 100 億美元的企業客戶在短短兩個月內翻倍，財星前十大企業有八家是他們的客戶。
​
在企業 LLM API 市場中，Anthropic 的市佔率高達 32%，反超 OpenAI 的 25%。現在每十個新客戶，就有七個選擇 Anthropic。
​
更可怕的是他們的旗艦產品 Claude Code。在 AI 程式設計工具領域，它拿下了 54% 的市佔率，單一產品年營收就超過 25 億美元，把 GitHub Copilot 和 Cursor 遠遠甩在後頭。企業一旦將這種高可靠性、低推論成本的工具整合進核心業務，幾乎就不可能替換了。
​
相比之下，OpenAI 高度依賴每月 20 美元的個人訂閱。ChatGPT 雖然有 9 億週活躍用戶，但 95% 是免費仔，龐大的推論成本根本無法被營收覆蓋。
 
當然，老大哥 OpenAI 不服氣。昨天，OpenAI 營收長 Denise Dresser 隨即發出一份內部備忘錄，直指這個數字有「水分」。
​
爭議點在於會計認列方式：Anthropic 把透過 Amazon 和 Google 轉售的營收以「毛額（Gross）」入帳（沒扣除通路分成），而 OpenAI 則是採「淨額（Net）」。
​
如果用相同標準計算，Anthropic 的實際 ARR 應該只有 220 億美元，依然落後於 OpenAI。
​
但撇開數字遊戲不談，底層的事實是：Anthropic正在追得十分貼近了，甚至可能已經超車。
​
▋從算力到防禦：Anthropic 的肌肉展現
​
除了營收，Anthropic 最近的動作頻頻，展現了極強的戰略底氣：
​
1. 算力保險： 他們剛與 Broadcom 及 Google 達成協議，確保了 2027 年起高達 3.5 GW（吉瓦） 的算力。這等於是提前買好了未來兩年的「戰爭門票」。
​
2. Project Glasswing： 他們發布了最新模型 Claude Mythos Preview，這款模型強大到能輕易找出全球作業系統中的數千個高嚴重漏洞。Anthropic 選擇不對公眾開放，而是組成一個包含 Apple、NVIDIA、摩根大通在內的「信任聯盟」，將最強的矛轉化為最強的盾。
​
▋現在說誰贏，都還太早
​
到了這個時刻，很多人開始說 Anthropic 已經贏得這場 AI 大戰了。
​
我的答案是：還早得很。
​
就在一年前，大家還在說 ChatGPT 一家獨大；半年前，Gemini 衝上第一時，很多人喊著 Google 要反超了；而現在，風向吹到了 Anthropic。
​
LLM 的競爭就像是一場沒有終點的馬拉松，而且每隔幾公里，跑道規則就會變一次。AI 技術的演進太快，現在的領先者不代表明天的贏家。我們正處於 AI 時代的「極早期」，與其急著選邊站，不如密切觀察這些巨頭如何調整他們的商業策略，從而作出投資判斷。
​
如果你想更完整地理解這場AI變革的細節，我建議你翻開最新一期的《商業周刊》。《商周》裡有很多高品質的深度分析（就像我寫的這篇，哈哈），絕對值得你買一份來讀。
