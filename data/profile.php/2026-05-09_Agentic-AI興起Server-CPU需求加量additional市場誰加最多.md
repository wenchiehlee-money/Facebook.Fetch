# Agentic AI興起Server CPU需求加量(additional)市場誰加最多?

Agentic AI興起Server CPU需求加量(additional)市場誰加最多?

AMD: 我市佔率竄起

Intel: 我有晶圓廠你們沒有

Nvidia: 我有Vera CPU機架和NVLink-C2C

ARM: 我有省電Arm架構包含四大CSP ASIC CPU和NV Vera以及自有產品AGI CPU

DRAM: 冷笑中......
你們知道馮紐曼架構嗎? 沒有additional的DRAM你們一顆CPU也出不了

Agentic AI興起之前DRAM就缺貨了，Agentic帶動CPU server(就是傳統通用server)之後，DRAM需求將搭配CPU數量而上升

x86搭配512GB~4TB DRAM推測1TB以上才能發揮多核心高階CPU效果
ARM CPU晶片客戶ASIC廣泛DRAM不定，Vera用1.5TB DRAM
