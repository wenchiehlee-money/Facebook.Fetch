# Mercury Research公佈1Q26 AMD CPU市占率，整理一下:

原文連結: https://www.facebook.com/permalink.php?story_fbid=pfbid0R4cVZ4Sm8BqDhbGs4SV4mmz1zH1HQRRo5zamjedkGheB3EnGvefxiLXX8rkH1vq3l&id=100054201473657
Mercury Research公佈1Q26 AMD CPU市占率，整理一下:

Server x86 CPU數量: 33.2% from 28.8%
Server x86 CPU營收: 46.2% from 41.3%
這兩個數字表示AMD的ASP比Intel高，這通常表示AMD每顆CPU的"核心數"(cores)比Intel多

Mobile x86 CPU數量: 28.3% from 26%
Mobile x86 CPU營收: 28.9% from 24.9%
表示AMD ASP雖然只有略高於Intel，但上季ASP低於Intel，提升很快

DT x86 CPU數量: 33.2% from 36.4%
(唯一市佔率季度衰退的產品線，但上一季是創紀錄)
DT x86 CPU營收: 37.6%
表示ASP ASP高於Intel

我覺得AMD在client端DT CPU和Mobile CPU的ASP高於Intel不奇怪，因為AMD強在consumer PC，Intel強在commercial PC，而commercial PC CPU相對比較低階，但是AMD的Server CPU不但市佔率屢創新高，ASP還加速提升

----------"AMD 佔伺服器 x86 CPU 營收的 46% — Intel 仍掌控 70% 的消費型 PC 市場份額

根據 Mercury Research 的數據，2026 年第一季對 AMD 來說相當有利，該公司成功提升了其在用戶端系統市場的佔有率，伺服器市場的佔有率也飆升至 33%。此外，公司在客戶端與伺服器市場的營收佔有率創下新高，因此 AMD 目前掌控了 x86 CPU 市場價值的 38.1% 以及 x86 伺服器 CPU 營收的 46.2%。或許一個令人擔憂的跡象是，公司桌上型電腦的單位與營收份額雖然年比年有所上升，但依序下滑。

消費級 CPU：AMD 取得進展，但幅度有限

在消費型 PC 領域，AMD 在 2026 年第一季持續攀升，其客戶端 CPU 佔有率提升至 29.6%，較 2025 年第四季的 29.2% 略有上升，且較去年同期的 24.1% 大幅提升，根據 Mercury Research 的數據。

然而，該季度顯示桌機與筆記型電腦市場出現分歧，英特爾成功回收了3.2%的桌上型電腦市場份額。此外，英特爾仍是客戶端CPU的主導供應商，佔有70.4%的市佔率，儘管其地位從2025年第一季的75.9%有所下滑，因為AMD在筆記型電腦市場表現相當不錯。

然而，在營收方面，AMD 的地位依然強勁。公司客戶端 CPU 營收佔有率達到 31.4%，略高於前一季，且遠高於去年同期的 26.6%，這或許反映出公司在高端客戶端處理器領域的持續強勢。儘管如此，Intel 仍掌控近 69% 的客戶端 CPU 營收，這是一件大事。今年下半年情況如何發展——當英特爾推出 Nova Lake 處理器，為客戶端系統寄予厚望——仍有待觀察。

桌上型 CPU：市佔率來了，市佔率又掉了

在桌上型電腦市場，AMD 回撤了部分在特別強勁的假日季度取得的巨大漲幅，但仍維持歷史高位，因此這次下跌可視為調整，而非新趨勢。

AMD 2026 年第一季桌上型電腦 CPU 佔有率為 33.2%，低於 2025 年第四季創紀錄的 36.4%，但遠高於前一季的 28%。英特爾隨後逐步追回部分市場，並將桌機市場佔有率提升至66.8%，但仍遠低於去年同期的72%，因為AMD持續保持比近年更強的地位。

在營收方面，儘管連續市佔率下滑，AMD 依然強勁：公司桌機 CPU 營收佔比為 37.6%，低於前一季創紀錄，但仍較去年同期高出 3.2%。Intel 持續創造了桌面 CPU 的大部分收入，但 AMD 能維持相對高收入佔有率的能力，顯示高階 Ryzen CPU 的持續強勢。

行動處理器：又創下一個創紀錄的季度

在行動電腦領域，AMD 再次展現了有史以來最強的成績，成功提升了市佔率，並創下筆電市場的最高市佔率。

AMD 行動處理器的佔有率在 2026 年第一季攀升至 28.3%，高於 2025 年第四季的 26%，也較去年同期的 22.5% 有所提升，創下公司行動處理器有史以來最佳季度表現。顯而易見，英特爾在市場中佔據了最大市場份額——71.7%，但隨著AMD透過提升可用性並擴大其在傳統由英特爾主導的細分市場（如商務與商用筆電）的佔有率，其領先優勢進一步縮小。

至於營收分成，AMD 的進展更為驚人。公司行動CPU營收佔有率提升至28.9%，較2025年第四季的24.9%及2025年第一季的22.2%有所提升，反映出高價值筆記型電腦處理器銷售強勁。Intel 持續掌控筆電 CPU 收入的多數（71.7%，低於 2025 年第一季的 77.5%），但 AMD 接近 28.9% 的營收份額，明顯顯示其在高利潤的高端筆電競爭日益增強，而這些筆電歷來幾乎完全偏袒 Intel 市場。

伺服器 CPU：又一個突破性季度

雖然第一季對 AMD 行動處理器來說表現不錯，但對於 AMD 用於伺服器的 EPYC CPU 則是例外。該公司不僅在單位佔有率上創下新高，還在單季營收佔有率飆升了5%。

Mercury Research 的數據顯示，AMD 的伺服器處理器佔有率攀升至 33.2%，較 2025 年第四季的 28.8% 及一年前的 27.2% 有所提升。英特爾仍以66.8%的份額出貨大多數伺服器處理器，但隨著EPYC採用在超大規模雲端供應商、企業部署及AI/HPC基礎建設中持續擴大，其地位在連續及年均有所削弱。

在營收方面，AMD 的表現更為突出：公司伺服器 CPU 收入佔有率創下 46.2% 的紀錄，這意味著 AMD 目前掌握近半數的 x86 伺服器 CPU 收入，且約佔三分之一的出貨。這份單位佔有率與營收佔有率的差距反映了 AMD 處理器整體平均售價顯著較高，以及公司高核心數量的高級配置的受歡迎程度。雖然英特爾的伺服器CPU收入高於AMD，但其Xeon產品的ASP低於EPYC，與前幾季的市場表現相當。

摘要
AMD 以強勁的姿態開啟 2026 年：根據 Mercury Research，擴大了其在客戶端與伺服器 CPU 的市占率，並創下整體 x86 CPU 營收佔有率的新紀錄。

公司在筆記型電腦與伺服器領域表現尤為強勁，EPYC 採用使 AMD 伺服器收入佔有率接近整個 x86 伺服器市場的一半。雖然 AMD 在特別強勁的假日季度後，桌上型電腦 CPU 佔有率連續下滑，但仍遠高於去年同期水準，因此公司強勁的動能持續存在。

整體而言，AMD 持續鞏固其在 x86 CPU 市場中最獲利的部分的地位，而 Intel 則維持出貨領先，但在營收佔有率與高級市場的市場則進一步失去。"
