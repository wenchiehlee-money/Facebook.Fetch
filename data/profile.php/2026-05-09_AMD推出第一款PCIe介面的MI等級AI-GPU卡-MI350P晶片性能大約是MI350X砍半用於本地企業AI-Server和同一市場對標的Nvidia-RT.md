# AMD推出第一款PCIe介面的MI等級AI GPU卡:  MI350P，晶片性能大約是MI350X砍半，用於本地/企業AI Server，和同一市場對標的Nvidia RTX Pro 6000 Blackwell Server Edition比較起來，表定硬體算力大致差不多，但VRAM大勝，AMD用144GB HBM3e，Nvidia用96GB GDDR6，兩家都沒有scale up都只有PCIe Switch慢速GPU to GPU通訊VRAM無法連成一大塊，這幾年本地推理Server市場一直沒起來，不知道銷售量如何，至少為簡單企業和教育應用，提供便宜的單卡到8卡的升級路線，以前這市場只有Nvidia，AMD目前MI350X系統全部是UBB scale up沒有PCIe，現在有PCIe市場有兩家可選擇AMD還多了HBM
