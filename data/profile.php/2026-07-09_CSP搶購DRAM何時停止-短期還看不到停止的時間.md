# CSP搶購DRAM何時停止? 短期還看不到停止的時間

原文連結: https://www.facebook.com/permalink.php?story_fbid=pfbid0T1ZQJZ2GEWBdp6fqwob33EmEJ1yW2GFcWr59boHasBTtv1ecD4GJxdQfD4FaAp7Wl&id=100054201473657
CSP搶購DRAM何時停止? 短期還看不到停止的時間

1. 根據TrendForce預估，3Q26 Server DRAM價格將上漲13~18% QoQ，略低於近日其他新聞來源報導的20~30%漲幅，中間的差距，可能來自於

(1)部分原廠提早於2Q26計入3Q26部分漲幅，漲幅pull-in ? 

(2)部分美系CSP因為已經簽了複數年份的LTA，這類客戶3Q26漲幅較低

因此，考慮了這兩點之後的產業平均漲幅約13~18%，如果有人採訪到沒有簽LTA的合約客戶、或者採訪到2Q26漲幅沒有特別高的原廠，問到的漲幅理應高於20%

2. 可以說，3Q26開始，同樣屬於合約市場客戶(大品牌/CSP等)，但其中的漲價幅度、報價價差，開始擴大，原廠server DRAM的出貨價格漲幅來源將逐步落在非長約客戶，LTA還是有用的(先樂後苦嗎?不知道)

3. 2Q26 CPU缺貨比DRAM嚴重，讓美系CSP長短料無法組裝成Server系統之下DRAM庫存升高，但CPU供應將於2H26~2027逐步增量下Server系統組裝增量下，將逐步消化DRAM庫存

4. 2027年整體DRAM將供不應求態勢確立，CSP普遍追加DRAM需求

5. 根據Late 2Q26各DRAM原廠2027年供給意向，RDIMM 2027全年供給僅增加15~20% YoY，和2027年Server CPU出貨預測比起來，Server DRAM供應增量大幅落後Server CPU增量，2027 Server DRAM繼續缺貨，對RDIMM買方而言，即使已確保2026下半年用量(庫存+commitment)，仍然會為2027供不應求繼續儲備庫存

6. 根據5，Server產業2Q26 CPU是最缺零件(短料)，到2027變成DRAM是最缺零件(短料)

7. 之前我們討論過DRAM產業供需最關鍵的一點: CSP無窮無盡的需求、大黑洞、大胃王，搶盡了原本供應給PC和手機的DRAM產能之後，何時能酒足飯飽停止搶料? 這個關鍵問題，根據5，CSP目前沒有停止搶購的跡象，將繼續為了2027年大缺貨(CPU>DRAM)儲備庫存(應該包含實體庫存和採購承諾commitment)

8. 根據7，手機品牌和PC品牌的DRAM採購......繼續頭痛，沒有淡季、沒有旺季、沒有pull in到1H26之後2H26的需求下降更好採購、沒有價格彈性讓需求下降的期待，因為大黑洞、大胃王CSP坐在你隔壁

9. 第一次看到調研很公開的資訊提到2027的各季價格趨勢預測，在考慮LTA追加量要加價，和非LTA客戶的漲價較多的因素之下，2H26~2H27整體Server DRAM合約價將逐季上漲，但漲幅收斂

9. 因CSP和OEM陸續調整RDIMM組合需求，部分自96/128GB下降至32/64GB，3Q26起低容量模組的出貨條數將出現季增，這點意味content per box可能下降? 或content per box上升的趨勢便緩和? 不知道，但這一點應該已經包含在總體供需結論裡面，不是未來的新變數，未來如果出現這類報導，要了解是(1)DRAM先不夠用只好降低content讓更多box出貨，而不是(2)content per box提升趨勢改變讓DRAM需求下滑，因果相反

------------"TrendForce: 長約定下漲幅天花板，預估3Q26 server DRAM合約價將季增13-18%

根據TrendForce最新記憶體價格調查，由於部分原廠提早於2026年第二季的報價反映漲幅，加上數家美系雲端服務供應商(CSP)已簽署複數年長約(LTA)，限制原廠對該類客戶的漲價空間，TrendForce預估第三季server DRAM合約價將季增13-18%。然而，隨著供不應求格局延續，後續仍可能出現各原廠競相上修報價的情況。

TrendForce表示，因CPU供給短缺影響server整機組裝節奏，美系CSP的DRAM庫存水準於第二季逐漸升高。不過，考量CPU供應將於2026下半年至2027年間逐步釋放、server整機組裝規模可望擴大，以及2027年server DRAM供不應求態勢已然確立，CSP普遍仍積極追加需求。

各記憶體原廠於第二季下旬陸續向客戶釋出2027年的供給意向，經TrendForce初步試算，合計原廠RDIMM位元供給年增幅僅約15-20%，將大幅落後server CPU顆數的成長。對RDIMM買方而言，即使已確保2026下半年的消耗量，仍有為2027年供不應求情勢儲備庫存的誘因。

部分CSP已和原廠簽署長約下，TrendForce分析，自2026年第三季起，原廠server DRAM的出貨價格漲幅來源將逐步落在非長約客戶上，以及透過非長約形式提供給長約客戶的追加供給。預計在2026下半年至2027下半年，整體server DRAM合約價將維持逐季上漲，而漲幅收斂。

觀察各server模組容量的供給組合，反映採購成本控管及CPU供給配套。自2026上半年以來，因CSP和OEM陸續調整RDIMM組合需求，部分自96/128GB下降至32/64GB。預期這項規格調降將於第三季起逐步於原廠的出貨組合實現，低容量模組的出貨條數將出現季增，原先出貨位元比重下降的情況也將趨緩。"
