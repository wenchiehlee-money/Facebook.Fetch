想知道 CPU 是否真的從被遺忘的角落重返舞台中央，看看最近亞馬遜 AWS 和Meta的消息就明白了。
​
在上星期發出的致股東信中，Andy Jassy表示，他們自研的 Graviton CPU（基於 ARM 架構）需求已達到白熱化程度。
​
最近有兩家不具名的大客戶，竟試圖直接買斷 AWS 在 2026 年「所有」可用的 Graviton 伺服器容量。這個請求最終被 AWS 拒絕，只為保護其他成千上萬的客戶。
​
另外，在幾個星期前，Meta與 NVIDIA 簽訂了數百萬顆處理器的多年協議，但其中最關鍵的佈局，是首次大規模部署「純 CPU」機櫃（採用 NVIDIA Grace CPU），完全不搭配 GPU。
​
Meta 的目標非常明確：專攻 AI 推理（Inference）、AI 代理（Agentic Workloads）和各種線上應用。
​
他們的盤算是，當模型訓練完成後，真正大規模、無時無刻不在運行的，是成本更敏感的「推理」任務。為此，他們正明確地擺脫「唯 GPU 論」，尋求更具成本效益的運算架構，而高效能 CPU 正是答案。
​
另外，微軟，Oracle，CoreWeave等也相繼部署了不同的純CPU數據中心，所有公司都在告訴大家，他們要開始搶CPU了。
​
