# 看到一份韓國DRAM、Flash、SSD的價格圖資料，火箭噴射，不禁想提醒一下，圖表是來自韓國政府海關資料，因為是政府海關進出口資料，不會精確統計，單位不是調研和一般常用的price per GB或Gb，而是price per KG，沒看錯，就是公斤，因為海關資料包山包海，只能用KG粗淺統計，如果是一般穩定的產品例如礦物、農作物......用公斤價格很OK，如果是一般產品價差大混在一起只要mix沒有每月變動，混在一起可能也OK，但是記憶體，一顆晶片或一條模組，4Gb和16Gb可能重量差不多，價格差四倍，模組32GB和8BG重量也差不多價格差四倍，SSD從1TB/4TB/8TB...128TB差異更大，所以在DRAM或SSD的 "容量增加"產品組合快速改變的時候，price per Kg是會加分的，price per Kg受到(1)漲價(2)高容量mix升級，兩個因素影響，不是只有(1)漲價一個因素

看到一份韓國DRAM、Flash、SSD的價格圖資料，火箭噴射，不禁想提醒一下，圖表是來自韓國政府海關資料，因為是政府海關進出口資料，不會精確統計，單位不是調研和一般常用的price per GB或Gb，而是price per KG，沒看錯，就是公斤，因為海關資料包山包海，只能用KG粗淺統計，如果是一般穩定的產品例如礦物、農作物......用公斤價格很OK，如果是一般產品價差大混在一起只要mix沒有每月變動，混在一起可能也OK，但是記憶體，一顆晶片或一條模組，4Gb和16Gb可能重量差不多，價格差四倍，模組32GB和8BG重量也差不多價格差四倍，SSD從1TB/4TB/8TB...128TB差異更大，所以在DRAM或SSD的 "容量增加"產品組合快速改變的時候，price per Kg是會加分的，price per Kg受到(1)漲價(2)高容量mix升級，兩個因素影響，不是只有(1)漲價一個因素

海關進出口資料用來分析總經比較適合，分析產業就要小心分辨定義和數字解讀
