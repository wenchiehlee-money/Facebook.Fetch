RBRK  (增加信念） 

Rubrik 攜手 Amazon Bedrock AgentCore，搶佔 AI Agent 安全治理市場。 

就在昨天的 AWS Summit，Rubrik  宣布其 Rubrik Agent Cloud（RAC）即將與 Amazon Bedrock AgentCore 整合。

Agentic AI 正從概念走向企業大規模部署，安全治理的缺口也隨之浮現。

企業在 AWS 上跑 AI Agent，面臨的挑戰是 Agent 對工具，對 LLM，甚至 Agent 之間的通訊，都可能成為攻擊或失控的入口。  傳統的靜態規則根本跟不上 Agent  的即時決策速度。

Rubrik 的解法是把偵測與執行分開處理， Rubrik 負責概率性偵測，AgentCore 的閘道層負責確定性執行， 兩者在 Agent 推理迴圈之外協作，讓每個 Agent 動作都能獲得一致的允許或拒絕決策。 

SAGE  是 Rubrik 的 AI 治理引擎，以意圖驅動的語義政策取代靜態人工監督。Agent Inventory則自動盤點跑在 AgentCore 上的所有 Agent，提供風險、權限與違規的全方位可視性。最後是統一控制平面，讓 IT 和資安團隊在同一介面管理 AI 政策與混合雲資料，不用跨平台切換。 

Rubrik AI GM Devvret Rishi 的說法很直接， 資安領導者估計，未來一年內他們面對的攻擊，有一半以上將由 Agentic AI 驅動。  換句話說，用來攻擊的是 AI，用來防守的也得是 AI。

這次與 AWS 的深度整合，是 Rubrik 在AI 安全賽道的明確卡位。 Rubrik Agent Cloud 的商業化能否轉化為可觀的 ARR 增長，將是接下來幾季財報值得追蹤的核心問題。
