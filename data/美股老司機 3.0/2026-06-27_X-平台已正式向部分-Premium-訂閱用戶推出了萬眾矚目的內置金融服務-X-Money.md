# X 平台已正式向部分  Premium+ 訂閱用戶推出了萬眾矚目的內置金融服務  X Money。

原文連結: https://www.facebook.com/reel/4486844768200407/
X 平台已正式向部分  Premium+ 訂閱用戶推出了萬眾矚目的內置金融服務  X Money。

X Money 簽帳金融卡（Debit / Flex）

這是一張極具質感的高級實體金屬 Visa 簽帳卡。卡片正面會印上用戶的 X 帳號與 ID。

支援全美/全球免費 ATM 提款，且免收海外交易手續費，預計還將提供最高 3% 的消費現金回饋。

業界領先利率， 用戶存入 X Money 帳戶的閒置現金將享有高達 6% 的活期年收益率。 該高利活存方案無最低存款餘額限制。

高達 1000 萬美元的 FDIC 存款保險，  傳統美國銀行的 FDIC 保險上限通常僅為 25 萬美元。而 X Money 透過其 X Cash Sweep Program，將資金分散存放於合作銀行中。

全方位支付， 無縫轉帳與日常消費。 整合了 Visa Direct 技術，用戶可以直接在 X 平台內向朋友進行點對點（P2P）即時轉帳，繳納房租，發送電匯（Wire）或郵寄紙本支票。

放棄了傳統密碼，全面改用更安全，更快速 Passkeys（通行鑰/生物辨識）進行解鎖與交易授權。

內建獨立的分帳管理頁面，包含日常消費帳戶，儲蓄帳戶以及商務帳戶，逐步落實馬斯克將 X 打造為萬能應用程式的金融藍圖。

目前 X Money 仍處於早期受控測試階段，優先開放給美國的部分 Premium+ 高階訂閱用戶搶先體驗，以便在全面公測前收集回饋。預計在 2026 年下半年，這項服務將會進一步擴大推向全美乃至全球的普通用戶。
