【#川普 貼文商品化，Truth Social 推即時 API】
​經營社群平台 Truth Social 的 Trump Media（DJT）週四宣布，將推出一項 API 服務，提供對 Truth Social 貼文的即時存取權限，讓客戶可以「早幾毫秒」看到對市場有影響力的貼文。
​
川普家族是 Trump Media 的最大股東，川普的帳號也是 Truth Social 上最大的帳號。雖然新聞稿中並未提及川普，但除了川普，很少有人能在 Truth Social 上持續提供影響市場的資訊。
​
因此這項 API 的主要商業價值，顯然就來自川普的市場影響力，由於川普的貼文經常引發金融市場劇烈波動，當付費客戶取得資訊時間差，就能在交易中搶佔先機。Trump Media 也在新聞稿中直言，市場已經會因 Truth Social 的貼文而波動。
​
Trump Media 透露，Truth API 將於 8 月 1 日啟動，並已簽約了一些機構客戶。
